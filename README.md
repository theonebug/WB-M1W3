# WB-M1W3
Графическая утилита на Python для настройки и мониторинга модуля WB-M1W2 v.3 (1-Wire → Modbus RTU). Позволяет подключаться по RS‑485, конфигурировать режимы входов, параметры опроса и детектирования нажатий, а также в реальном времени отслеживать показания до 20 датчиков на каждом из двух входов.


## Возможности

* **Подключение** через COM‑порт (RS‑485) с настройкой скорости, чётности, стоп‑битов и Modbus‑адреса.
* **Чтение и запись конфигурации**:

  * Режим работы каждого входа: 1‑Wire или «Счёт и детектирование нажатий».
  * Период опроса датчиков (1–60 с).
  * Коэффициент фильтрации выбросов (для отсеивания ошибочных значений 85°C).
  * Время подавления дребезга (0–100 мс).
  * Параметры распознавания нажатий: длительность длинного нажатия, окно двойного нажатия.
* **Мониторинг в реальном времени**:

  * Таблица на 40 строк (20 датчиков на вход 1 + 20 на вход 2).
  * Отображение температуры, статуса подключения («OK» / «Нет датчика»), счётчика нажатий (для дискретного режима).
  * Автоматическое обновление с заданным интервалом (1–60 с).
* **Совместимость с разными версиями прошивки**:

  * Если устройство имеет прошивку ниже 4.35.0 (без расширенных регистров), программа автоматически переключается на базовые регистры (температура только первого датчика на каждом входе).
